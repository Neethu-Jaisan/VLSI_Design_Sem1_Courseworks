# CMOS Full Adder using LTspice

This project implements a CMOS full adder using basic logic gates and simulates
its digital operation using LTspice.

## Tool
- LTspice

## Features
- XOR-based sum generation
- CMOS AND-OR carry logic
- Transient verification of truth table
